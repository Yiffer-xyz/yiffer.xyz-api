Hot tips

In the first two lists, click any element and begin typing a name/tag to search for it, like any normal dropdown anywhere.
In the "tags to add" list, you can press enter when having a tag selected, which will do the same as pressing the blue "Add".
Use the two "tips" above to speed up the tagging-adding.

Here are some things that are commonly forgotten:
  • Always tag the name for any pokemon, digimon, pony, etc!
  • - Every comic should be either anthro, feral, or both!
  • - When adding a species like cat, horse, pony, dog, be sure to also include the family, in these cases feline, equine, equine, canine
  • - Consider adding story or long story tags if there's a story to the comic
  • - Size difference is often forgotten
  • - There's no need for a MLP or my little pony tag, as that's already in the category feature along with furry, pokemon, etc

Other notes: Tag characters that are recognizable. Examples: Fox McCloud, Falco Lombardi, etc; Bowser, Isabelle from Animal Farm, Kindred from League of Legends... any pokemon. The main ponies from MLP. Basically just don't tag OCs.
If you're not certain whether a tag should be there or not, e621's tagging system is great for guidance. Remember, however, that we don't want to be as detailed in the tagging as e621. Common tags over there like "clothing", "paws", "solo", "butt" aren't needed here. Also, looking at already tagged comics might help. The first comics alphabetically are all tagged.

In the end, our only goal is to help users find what they want! So if you're wondering if something should be tagged as X or not, think along the lines of "if I were a user looking for comics with X in them, would I want this comic to appear in the list?"

If you have any questions, just ask in the Discord chat! I promise I'll never be angry with anyone for asking questions!

Creating tags

{{newTagSentMessage}}

You may add a new tag, but please take a moment to think through this. Being accurate is good, but being obsessive about detail is not! Example: the tag "sex toy" is enough. There is no need for "vibrator", "dildo", "strap-on". We're dealing with comics, not an image database (in which details are good, due to a much bigger selection) That said, there are definitely many, many tags not in the list, so don't be afraid to add new ones. Better having too many than too few, so add away!
